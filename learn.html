Front-End Web Development

So, someone(me) probably sent you here because you asked how to become a web developer. It's not an easy question so I thought I'd throw together this little site. I'll include services/products I use regularly and little tidbits I've picked up along the way.

I found web development through a coding camp my parents workplace offered in around mid 2019. We created an anti-bullying website so we were able to learn the basics of html/css/js. Once the site was done the camp finished and I continued learning on my own for a bit learning via Udemy and creating my first portfolio website before leaving development behind for a long time.

I started again in October of 2020, made a website for The Fundementals and from there my prices went up and my effort up too. I learned I really love doing frontend and I can't wait to get into backend. Also, Tailwind is amazing, never learn Bootstrap. :)
